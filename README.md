    ->https://youtu.be/UgX5lgv4uVM?si=E9M8iXKnMcBGqE9u

![img.png](img.png)

To see all HTTP requests go to content.http file in the root directory.

Concepts learned:
    - Spring Core
    - Spring MVC
    - Spring Data 
    - MySQL

To run application in terminal:
    - mvn clean install
    - mvn spring-boot:run
